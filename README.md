## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## How to set up

```bash
# generate migrations
$ npm run migration:generate
```
```bash
# run migrations
$ npm run migration:run
```
```bash
# run seeds 
$ npm run seeds:run
```
